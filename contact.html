Found bugs or errors? You would like to have special enhancements?

Please open an issue at the corresponding repository in GitHub describing the error in detail, so we can reproduce it and fix it. We would also like to enhance the capabilities of primer3, but we need good descriptions for suggested improvements and the problem they are intended to solve.

Made modifications or enhancements to primer3, primer3web or primer3plus?

We would be happy to enhance the main distribution. Please contact us to arrange code exchange or submit a pull request at GitHub.

Problems using primer3, primer3web or primer3plus?

We do not have the resources to provide end-user support. Nevertheless, you can send us your problem description and we will update the manual at this web site to address this problem.
